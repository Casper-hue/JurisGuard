# JurisGuard

JurisGuard是一个专为法律专业人员设计的智能合规管理平台，集成了AI助手、法规追踪、风险评估和文档管理功能。

## 🎯 项目亮点

### 🤖 AI驱动的智能合规引擎
- **先进AI模型集成**：支持OpenRouter、Hugging Face、Together AI等主流大语言模型
- **专业法律分析**：模拟资深法务专家（15+年跨国企业合规经验）进行法规解读
- **精准风险识别**：自动扫描法律文本，识别潜在合规风险点并提供应对策略
- **多法域覆盖**：涵盖GDPR、EU AI Act、CCPA等全球主要法规体系

### ⚡ 现代化Web技术栈
- **Next.js 16 + React 18**：高性能SSR/SSG，极致用户体验
- **TypeScript强类型**：提升代码质量与维护性
- **Tailwind CSS**：美观响应式UI，完美适配多设备
- **Framer Motion**：流畅动画效果，增强交互体验

### 🔄 智能数据处理流水线
- **端到端数据流**：用户输入 → AI分析 → 结构化提取 → 数据持久化 → 可视化展示
- **统一数据模型**：Cases和Regulations标准化格式，确保数据一致性
- **自动化ID生成**：为每条记录创建唯一标识符
- **容错处理机制**：完善错误捕获与用户友好的反馈系统

### 🚀 核心功能模块

#### 📊 智能合规仪表板
- **实时监控**：动态显示合规警报与风险指标
- **可视化分析**：交互式风险雷达图与多维数据透视
- **智能筛选**：按地区、法规类型、风险等级灵活过滤
- **关键指标面板**：一目了然掌握整体合规状况

#### 🤖 AI合规专家助手
- **专业角色设定**：模拟资深法务专家思维模式，提供权威解答
- **法规查询引擎**：快速检索全球法规，支持跨法域对比
- **文档智能解析**：上传法律文档，自动提取关键信息与风险点
- **结构化输出**：遵循法律科技行业标准的回答格式

#### 📋 全方位合规管理
- **法规追踪系统**：实时更新欧美、东南亚重要法规，支持GitHub Actions自动化爬取
- **案例知识库**：丰富的法律案例与判例分析，支持智能检索
- **文档中心**：合同模板、政策文件集中管理，版本控制与权限管理
- **风险量化评估**：基于财务、法律、业务等多维度的综合风险评分

## 🚀 快速开始

### 环境要求
- Node.js 18+
- npm 或 yarn
- 支持的大语言模型API密钥（可选）

### 安装步骤

1. 克隆项目
```bash
git clone https://github.com/Casper-hue/JurisGuard.git
cd JurisGuard-Gemini
```

2. 安装依赖
```bash
npm install
# 或
yarn install
```

3. 配置环境变量（可选）
创建 `.env.local` 文件：
```env
NEXT_PUBLIC_AI_API_KEY=your_api_key
NEXT_PUBLIC_AI_BASE_URL=https://api.example.com/v1
NEXT_PUBLIC_AI_MODEL=model_name
```

4. 启动开发服务器
```bash
npm run dev
# 或
yarn dev
```

5. 访问应用
打开浏览器访问 http://localhost:3000

## 🔧 配置说明

### AI助手配置
在AI助手页面点击设置按钮，配置：
- API密钥: 支持多种大语言模型API

- 参数调优: Temperature、Max Tokens等

### 数据源配置
项目使用JSON文件存储静态数据，支持：
- 合规警报数据 (data/compliance-data.json)
- 案例数据 (data/cases-data.json)
- 文档数据 (data/documents-data.json)
